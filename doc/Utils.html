Class: Utils

Utils()

new Utils()

Utils provides miscelaneous useful methods

This class doesn't need to be instantiated, as all its methods are static in order to make those utils methods available without constraints. Refer to each method for their associated documentation. This class contains several methods to manipulate canvas' contexts or make color conversions.
